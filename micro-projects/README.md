# 🧪 Micro Projects

A collection of micro-project ideas for daily deliberate practice.
Target: each project finished in 1-3 hours.

## How to use

1. Pick a project that matches what you're currently learning (or go random)
2. Time-box 1-3 hours max — if you go over, shrink the scope
3. Build it, then capture insights & struggles in your daily learning log
4. Mark `- [x]` here once done
5. Update the progress counter below

## Rules

- 1 project = 1 concept focus (don't cram multiple theories at once)
- Feel free to redo the same project with a different stack/approach
- "Done" means you can explain it to yourself without looking at notes
- Capture takeaways in the learning log on the same day
